## Gestión de personal

Software para el departamento de Recursos Humanos para poder gestionar los datos de los distintos empleados de la empresa y de los currículumns que llegan a la empresa.

Algo que destacar, es la presentación de los datos, a partir de la presentación que hace Microsoft Outlook de los contactos por tarjetas de visita, se optó por presentar los distintos trabajadores en tarjetas de visita, en la que de un vistazo podías ver de cada uno los datos elegidos por el departamento (nombre, apellidos, teléfono de contacto, correo electrónico, categoría, etc), con opciones de búsqueda rápida para situarse en una ficha.

Para entrar en la gestión de cada persona (sea empleado o no) se hacía doble click y a partir de ese momento se abrian las distintas opciones que se podían realizar.

Lenguaje de programación - Delphi.
Base de datos - SQL Server

## Marketing: sistema de catálogos, folletos...

Para el departamento de ventas era importante poder sacar un folleto cada cierto tiempo, relativamente corto. Además de poder sacar un catálogo ya más alargado en el tiempo. Además de poder gestionar los datos de los artículos que verían los clientes en la plataforma de venta on-line. 

Para esto se dotó a distintas personas encargadas de esta labor de unas herramientas que permitían gestionar desde la carga de descripciones, asignació de imágenes, datos técnicos, etc. Hasta el trabajo de exportación de un catálogo a formato PDF que se dividía de forma automática en archivos de un número de paginas para no saturar la memoría del equipo y por lo tanto perder la impresión posterior. 

También se generó un proceso de exportación de datos para que [Adobe Illustrator](https://www.adobe.com/es/products/illustrator.html) pudiera importar los datos en formato XML y facilitar al diseñador la carga de los datos e imágenes para su procesado posterior.

Por último, para poder verificar que no quedaban datos por _arreglar_ se elaboró un sistema de consultas que permitían la modificación en grupo de datos de forma rápida, para ayudar a los usuarios a dejar los datos de la mejor forma para su posterior publicación.

Lenguaje de programación - Delphi
Base de Datos - SQL Server, DB2
Comunicación con Software de Terceros - Adobe Illustrator

## Impresión: integración con Microsoft Office

En el software de Gestión que desarrollabamos, se requería poder usar Microsoft Word, como gestor de informes. El proyecto era interesante, principalmente por la poca documentación que había en ese momento sobre la interacción con el paquete de ofimática.

Se creó un _gestor de informes_ que usaba la combinación de correspondencia para insertar los campos en una plantilla de Word, dicha plantilla posteriormente se usaba para imprimir informes, recibos, facturas, etc.

De esta forma, se incluyó un gestor de informes en un software de Gestión, que permitía al cliente usar su paquete ofimático.

Lenguaje de programación - Delphi
Base de datos - FireBird.
Comunicación con Software de Terceros - Microsoft Office
